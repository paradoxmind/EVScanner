# EVScanner -- Environment Scanner
Assembly that reads the host machine's runtime environment for error logging or diagnostic decisions. 

The following data is collected:

    OSDescription
    OSArchitecture
    FrameworkDescription

    CPUCount
    RunningProcesses
    
    MemoryInfo
        *Total Memory
        *Available Memory
    
    DiskInfo
        *Drive Name(s)
        *Total Size GB
        *Free Space GB
